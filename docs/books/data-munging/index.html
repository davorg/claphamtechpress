Data Munging with Perl

Techniques for data recognition, parsing, transformation and filtering

By Dave Cross

Coming soon

Data Munging with Perl teaches practical techniques for recognising, parsing, transforming and filtering data with Perl. It shows how to break munging problems into manageable stages, design useful data structures, emulate Unix-style filters and build parsers for complex formats including HTML and XML.

About Dave Cross

Dave Cross has been working in IT since 1988. Although he is best-known as a Perl programmer, he has experience in a large number of technologies including databases, web architecture and source code control. His interests in source code control and CI/CD have combined to create his most recent book, "GitHub Actions Essentials".
